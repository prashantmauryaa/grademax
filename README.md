# GradeMax AI

An ethical academic optimization platform that helps college students understand syllabus structures, exam patterns, attendance rules, and grading systems to optimize effort and improve CGPA intelligently.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: CSS Modules + Vanilla CSS Variables (No Tailwind by default, transitioned partially)
- **Animations**: Framer Motion
- **Language**: TypeScript

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Features implemented
- **Landing Page**: High-conversion hero section with glassmorphism.
- **Onboarding Flow**: Multi-step form with animations and Data Input step.
- **Dashboard**: Strategic ROI Matrix (Bubble Chart), Focus Mode, Risk Alerts.
- **Timeline**: Semester study visualizer.
- **Attendance**: Safe/Risk zone visualization.
- **Risks**: Predictive warnings panel.
